School Projects

All sites have responsive design and can be viewed on most technology

Bamazon App

A command line app that uses node.js to simulate an online shopping database; uses MySQL to store all data. In order to look at this project you will need to fork the repository over to your ccomputer or watch the demo video to see functionality.

Hangman Game

A simple Mario themed hangman game made with vanilla javascript. Designed using Bootstrap and local CSS.

Friend Finder App

An Express app developed to help pair users with someone who has similar personality traits as them.

Trivia Game

A Nintendo trivia game built with JavaScript and jQuery.Designed using Bootstrap and local CSS.

Giftastic Gifs

A site that uses JavaScript, jQuery and AJAX to provide the user with search related Gifs. This application uses local storage to store individual searches and saves searched buttons. Designed using Bootstrap and local CSS.

Train Time Tracker

A Firebase supported logging system that takes in multiple user data and stores it on Firebase servers and provides simple calculated response. Designed using Bootstrap and local CSS.

Liri Node App

A command line app that uses node.js to access multiple API databases. In order to look at this project you will need to fork the repository over to your computer and put your own access keys in a ".env" file.

Eat the Burger

An application that allows users to enter burger related data that is stored in a MySQL database through an Express server. Uses node.js, handlebars, Bootstrap, Javascript, jQuery, CSS and all associated npm packages. Deployed through heroku sites.
